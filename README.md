# Poline(ポリーヌ)

## 今回のポートフォリオのコンセプト
大阪にある、女性向けのインテリア商品を中心に取り扱う雑貨店のサイト。


## ペルソナ
 - どんな人に伝えたい？：10,20代の若い女性、雑貨やインテリアが好きな女性
 - 何を伝えたい？：お店の雰囲気や世界観、お店の情報、新入荷などのニュース
 - なぜ伝えたい？：お店に足を運んで欲しいから、お店の雰囲気を感じて欲しいから
